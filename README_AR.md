# أدوات المطابقة والرقابة المالية

تطبيق PWA عربي يجمع ثلاث أدوات في مشروع واحد:

1. التسوية البنكية.
2. مطابقة حسابات العملاء والموردين.
3. الموازنة مقابل الفعلي.

## صيغ الملفات

- Excel: XLSX / XLS
- CSV
- PDF نصي
- PDF ممسوح ضوئياً عبر OCR اختياري

## التشغيل

يمكن فتح التطبيق من GitHub Pages أو تشغيله من خلال أي خادم محلي ثابت.

## ملاحظات PDF

يستخرج التطبيق النص من PDF النصي مباشرة. في الملفات الممسوحة، يمكن تشغيل OCR، ثم مراجعة الصفوف وتعيين الأعمدة قبل الاعتماد. دقة الجداول تعتمد على جودة الملف وترتيبه.
