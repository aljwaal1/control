# أدوات المطابقة والرقابة المالية

تطبيق PWA عربي يجمع ثلاث أدوات في مشروع واحد:

1. التسوية البنكية المتقدمة.
2. مطابقة حسابات العملاء والموردين.
3. الموازنة مقابل الفعلي.

## التسوية البنكية V1.1

- رفع دفاتر الشركة وكشف البنك.
- رفع ملف مستقل لعمليات تسوية الشهر السابق.
- تمييز جهة العملية المرحلة: دفاتر الشركة أو كشف البنك.
- تسوية العمليات القديمة التي ظهرت في الشهر الحالي قبل مطابقة العمليات الجديدة.
- استمرار ترحيل العمليات التي لم تظهر بعد.
- تصنيف تلقائي للفروقات، مثل الشيكات المعلقة والإيداعات بالطريق والعمولات والفوائد.
- إظهار سبب عدم المطابقة لكل عملية.
- مذكرة تسوية تفصل الفروقات الحالية عن الفروقات المرحلة.
- منع استخدام العملية نفسها أكثر من مرة.

## صيغ الملفات

- Excel: XLSX / XLS
- CSV
- PDF نصي
- PDF ممسوح ضوئيًا عبر OCR اختياري

## التشغيل

يمكن فتح التطبيق من GitHub Pages أو تشغيله من خلال أي خادم محلي ثابت.

## ملاحظات PDF

يستخرج التطبيق النص من PDF النصي مباشرة. في الملفات الممسوحة، يمكن تشغيل OCR، ثم مراجعة الصفوف وتعيين الأعمدة قبل الاعتماد. دقة الجداول تعتمد على جودة الملف وترتيبه.
